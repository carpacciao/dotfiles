# Arch

## OS

Prefered OS:
- [EndeavourOS](https://endeavouros.com/)


## EndeavourOS

- Installed with KDE Plasma desktop
- Install things with
    - `sudo pacman -S <package>`
    - `yay <package>`
 
- `timedatectl set-local-rtc 1 --adjust-system-clock` disables wrong time windows/linux
